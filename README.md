**imgResize.py**
>Python代码 (Windows)

>根据指定宽度批量修改图片文件(jpg\gif\png)的尺寸, 保持比例, 统一另存为jpg文件

>使用\*.bat文件调用, \*.bat文件在Win+R里可不写扩展名调用, 把\*.bat和imgResize.py放在同一文件夹并加入环境变量

>例如 to700.bat
```
@echo off
change-img-size.py 700
```
>执行后提示
```
img0.doNothing--------(700width.jpg)
img1.resize.gif2jpg---(quanquan.gif)
请按任意键继续. . .
```
